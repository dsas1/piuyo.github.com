Data and Data Architecture Strategy

Data and Data Architecture strategy has five components that guide the leader

  • Data requirements

  • Data and Data Architecture

  • Technology component

  • Skills and capacity

  • Support for machine learning

There are three alternatives for data science infrastructure

  • fully built and maintained in house
  • hosted via managed services using cloud service providers
  • reliance on commercial solutions.

Components of a Data and Data Architecture Strategy

  • Data Requirements
    • What data should be included in the Data Strategy?
  • Data Governance
    • How will we promote data availability?
    • How will we promote usability?
    • How will we guarantee integrity?
    • How will we guarantee security?
  • Technology
    • What are the components of your Data Architecture?
  • Skills and Capacity
    • How will we promote the development of data literacy skills and capacity throughout the organization?
  • Support for Machine Learning
    • What are the components of the machine learning architecture and how will it interface with the data architecture?
